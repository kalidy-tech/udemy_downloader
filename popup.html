How to Use

• Login to Udemy Account

You must have a Udemy account and connect it with this extension.

• Turn On Developer Mode

Go to More Tools → Extension → Turn on Developer Mode

• Save As Screen

If you disable the "Ask where to save before downloading" feature in Chrome settings, you can download videos in an organized manner.
Change this setting at: chrome://settings/downloads

• Download Folder Structure

Videos will be saved in the following structure:
Your Download Folder \ Udemy Downloads \ Instructor Name \ Course Name \ Video Name.mp4
Example: C:\Users\PC1\Downloads\Udemy Downloads\JosePortilla\CompletePython\1.CourseIntroduction.mp4

• My Videos Are Missing

Only video content is analyzed. The extension does not download articles or files.
